Getting Started

Welcome to Fourth Dimension Web Analyzer

Our platform helps you analyze websites comprehensively with tools for link analysis, SEO optimization, content analysis, and more.

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Stable internet connection
  • JavaScript enabled
  • Valid email address for account creation

First Steps

  1. Sign Up: Create your account with a valid email address
  2. Verify Email: Check your inbox and click the verification link
  3. Choose Subscription: Select a plan that fits your needs
  4. Start Analyzing: Begin using our powerful analysis tools

Account Management

Profile Management

Update your personal information, change passwords, and manage account settings.

  • Navigate to Profile from the main menu
  • Update your name and other details
  • Change password (requires current password)
  • View account activity and login history

Password Security

Your password must meet these requirements:

  • At least 8 characters long
  • Contains uppercase and lowercase letters
  • Includes at least one number
  • Has at least one special character (!@#$%^&*)

Email Verification

If you haven't verified your email or need to resend verification:

  1. Go to the login page
  2. Click "Resend Verification"
  3. Enter your email address
  4. Check your inbox for the verification link

Subscriptions & Token System

Understanding Tokens

Our platform uses a token-based system. Each analysis tool consumes a certain number of tokens. You get daily tokens with your subscription, and can purchase additional tokens when needed.

Subscription Plans

Visit the Subscriptions page to view available plans and features. Each plan includes:

  • Daily token allocation
  • Access to all analysis tools
  • Search history and exports
  • Email support

Token Usage by Tool

Analysis Tool Token Cost Description
URL Search 1 Token Basic link analysis and robots.txt checking
Keyword Analysis 3 Tokens In-depth keyword density and content analysis
Image Analysis 2 Tokens Extract and analyze all images from a page
Heading Analysis 1 Token H1-H6 tag structure analysis
Meta Tag Analysis 2 Tokens SEO meta tags and schema markup analysis
Site Structure 2 Tokens Comprehensive website crawling and visualization

Purchasing Additional Tokens

  1. Go to your Subscriptions page
  2. Click "Purchase Additional Tokens"
  3. Select the number of tokens you need
  4. Complete payment via Razorpay
  5. Tokens are immediately added to your account
Token Expiration

Additional tokens expire when your current subscription ends. Daily tokens reset every 24 hours.

Analysis Tools Overview

Fourth Dimension provides six powerful analysis tools to help you understand and optimize websites:

URL Search 1 Token

Analyze all links on a webpage, categorize internal vs external links, and respect robots.txt rules.

Keyword Analysis 3 Tokens

Extract text content and analyze keyword density, frequency, and distribution across the page.

Find all images on a page with details like file size, dimensions, alt text, and optimization status.

Examine the heading structure (H1-H6) for SEO optimization and content hierarchy.

Comprehensive website crawling with visual representation of site architecture.

URL Search Tool

The URL Search tool analyzes all links found on a webpage and categorizes them as internal or external links.

How to Use

  1. Navigate to URL Search from the main menu
  2. Enter the website URL you want to analyze
  3. Choose whether to respect robots.txt rules
  4. Click Analyze to start the process
  5. Review the results showing internal and external links

Understanding Results

  • Home Links: Internal links within the same domain
  • External Links: Links pointing to other domains
  • Robots.txt Status: Shows which links are allowed/disallowed

Export Options

Download results as CSV file containing:

  • All discovered links
  • Link types (internal/external)
  • Robots.txt compliance status

Keyword Analysis Tool

Perform in-depth keyword analysis to understand content focus and SEO optimization opportunities.

How to Use

  1. Go to Keyword Search
  2. Enter the URL to analyze
  3. Click on any link from the results
  4. Enter up to 10 keywords (comma-separated)
  5. Click Analyze Keywords

Analysis Features

  • Keyword Density: Percentage of content containing each keyword
  • Frequency Count: Number of times each keyword appears
  • Text Extraction: Full text content from the page
  • Content Correction: Suggestions for improving text quality

Best Practices

  • Focus on 3-5 primary keywords per page
  • Aim for 1-3% keyword density
  • Use variations and related terms
  • Ensure keywords appear in headings and meta tags

Image Analysis Tool

Discover and analyze all images on a webpage for SEO and performance optimization.

How to Use

  1. Access Image Search from the menu
  2. Enter the webpage URL
  3. Analyze to get the link list
  4. Click on any link to extract images
  5. Review detailed image information

Image Information Provided

  • Image URL: Direct link to the image file
  • Alt Text: Alternative text for accessibility
  • Title Attribute: Additional image description
  • File Size: Image file size for performance analysis
  • Dimensions: Width and height in pixels
  • File Extension: Image format (jpg, png, webp, etc.)

SEO Optimization Tips

  • Always include descriptive alt text
  • Optimize image file sizes for faster loading
  • Use appropriate file formats (WebP for modern browsers)
  • Include keywords in image filenames

Heading Analysis Tool

Analyze the heading structure (H1-H6 tags) of web pages for SEO and content organization.

How to Use

  1. Navigate to Heading Search
  2. Enter the URL to analyze
  3. Select a link from the results
  4. View the complete heading hierarchy

Analysis Results

  • Heading Hierarchy: Proper H1-H6 structure
  • Heading Count: Number of each heading type
  • Content Length: Character count for each heading
  • SEO Compliance: Heading best practices check

SEO Best Practices

  • Use only one H1 tag per page
  • Keep H1 tags under 60 characters
  • Follow logical hierarchy (H1 → H2 → H3)
  • Include target keywords in headings

Meta Tag Analysis Tool

Extract and analyze meta tags, Open Graph data, and structured data for SEO optimization.

How to Use

  1. Go to Meta Search
  2. Enter the webpage URL
  3. Analyze to get available links
  4. Click on any link to examine meta data
  5. Review all meta information

Meta Data Extracted

  • Title Tag: Page title for search results
  • Meta Description: Page summary for search snippets
  • Meta Keywords: Keyword meta tags (if present)
  • Open Graph Tags: Social media sharing data
  • Twitter Cards: Twitter-specific meta data
  • Schema Markup: JSON-LD structured data

Meta Tag Optimization

  • Title tags: 50-60 characters optimal
  • Meta descriptions: 150-160 characters
  • Include target keywords naturally
  • Use unique titles and descriptions per page
  • Add Open Graph data for social sharing

Site Structure Tool

Perform comprehensive website crawling to understand site architecture and discover all pages.

How to Use

  1. Access Site Structure from the menu
  2. Enter the website URL to crawl
  3. Click Start Crawling
  4. Monitor progress on the loading page
  5. View the visual site structure when complete

Crawling Process

The crawler will:

  • Start from the provided URL
  • Follow all internal links
  • Respect robots.txt rules
  • Check HTTP status codes
  • Build a complete site map

Results Visualization

  • Site Map: Visual representation of site structure
  • Page Status: HTTP response codes for each page
  • Link Relationships: How pages connect to each other
  • Broken Links: Pages returning error codes
Crawling Time

Large websites may take several minutes to crawl completely. The process runs in the background and shows real-time progress.

Troubleshooting

Common Issues and Solutions

Analysis Not Working

Possible Causes:

  • Website blocks automated crawling
  • Invalid or non-existent URL
  • Insufficient tokens remaining
  • Network connectivity issues

Solutions:

  • Verify the URL is correct and accessible
  • Check your token balance
  • Try a different URL to test functionality
  • Contact support if issues persist

Access Denied Errors

Common Causes:

  • Subscription expired
  • Daily token limit reached
  • Account not verified

Solutions:

  • Check subscription status
  • Purchase additional tokens
  • Verify your email address
  • Renew expired subscription

Slow Performance

Optimization Tips:

  • Analyze smaller sections of large websites
  • Use specific pages rather than entire domains
  • Close unnecessary browser tabs
  • Check your internet connection speed

Browser Compatibility

For best performance, use:

  • Chrome 90+ (Recommended)
  • Firefox 88+
  • Safari 14+
  • Edge 90+

Contact Support

If you continue experiencing issues:

  1. Visit the Contact page
  2. Provide detailed description of the problem
  3. Include the URL you were analyzing
  4. Mention your browser and operating system
